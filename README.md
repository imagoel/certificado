# Gerador de Certificados

Aplicação web simples para gerar certificados personalizados no navegador.

## Como usar

1. Abra `index.html` no navegador.
2. Preencha nome, curso e data.
3. Clique em **Gerar Certificado**.
4. Clique em **Baixar PNG** para salvar a imagem.

## Arquivos

- `index.html`: estrutura da página
- `styles.css`: estilos da interface
- `app.js`: lógica de geração e exportação
